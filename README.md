# portfolio-platform
A production-grade personal portfolio platform built with React, Spring Boot Microservices, Spring Cloud Gateway, JWT Authentication, PostgreSQL, Redis, Docker, and CI/CD.

## Tech Stack

- Java 17
- Spring Boot 3.5
- Spring Cloud
- Eureka Server
- Spring Cloud Gateway
- Spring Security
- JWT
- MySQL
- Redis
- React + Vite
- Docker
- GitHub Actions

## Services

- ✅ Eureka Service
- ⏳ Gateway Service
- ⏳ Auth Service
- ⏳ Portfolio Service
- ⏳ Contact Service
- ⏳ Analytics Service
- ⏳ Notification Service
